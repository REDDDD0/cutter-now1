# Cutter Now
🎬 Cutter Now – Download and Trim YouTube Videos Easily

Cutter Now is a simple and elegant web app that lets you download any YouTube video in your preferred format (MP3 or MP4). You can also select a specific portion of the video to extract and download—perfect for creating clips, highlights, or audio snippets.

Key Features:

    🔗 Paste a YouTube video link and instantly preview it

    🎞️ Choose from multiple video and audio formats

    ✂️ Trim the video by selecting a start and end time

    ⬇️ Download your selected content with one click

No registration required. 100% free and ad-ready!

```bash
npm install
npm start
